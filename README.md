StockPulse is a Dead Stock Risk Prediction and Recovery System designed to help businesses identify at-risk inventory and take timely recovery actions.

## Features
- Secure Login System
- Inventory Risk Dashboard
- Dead Stock Identification
- Product Risk Explainability
- Sales & Inventory Analytics
- Report Export (CSV, Excel)

## Tech Stack
- Python
- MariaDB (XAMPP)
- Tkinter
- Pandas
- Matplotlib

## Project Structure
The project follows a modular structure separating authentication, database access, analytics, risk logic, and UI components.
